# aiddata-visualization
This project visualizes global foreign aid flows using data from AidData.
It includes an interactive world map, bar charts, and filters to help users explore trends in aid received, aid donated, and net aid flow across countries.

**1. Interactive World Map**
- Displays total aid received/donated by each country
- Color-encoded values
- Tooltips showing detailed statistics
- Zoom & hover effects
- Fully responsive

**2. Bar Chart**
- Top 10 countries by net flow (donated − received)
- Bars ordered from largest → smallest
- Positive values mean receiving more
- Negative values mean donating more
- Animated transitions

**3. Filter Controls**
- Dropdown filters (e.g., purpose categories)
- Year selection
- Dynamic updates to both visualizations

**Technologies Used**
- D3.js v7
- TopoJSON
- HTML / CSS / JavaScript

**Data Source**
- The dataset comes from AidData, a research lab at William & Mary.
